# Home Assistant Add-ons

Custom Home Assistant add-ons by covertconvert.

## Installation

Add this repository to your Home Assistant instance:

1. Open Home Assistant → **Settings** → **Apps** → **Install apps** (blue button)
2. Click the ⋮ menu in the top right → **Repositories**
3. Paste this URL: `https://github.com/covertconvert/home-assistant-addons`
4. Click **Add**, then refresh the store

The add-ons will appear under "Home Assistant Add-ons".

## Add-ons in this repository

### [Claude Code Messages](./claude_code_messages)

Graphical chat interface for the Claude Code CLI, designed for the Home Assistant companion app. See its [README](./claude_code_messages/README.md) for details.
